To access the assignment:
- please install Node.js from the web: go to nodejs.org and follow the instructions
- once done, please install npm
- then, install express, cookieParser, and bodyParser
- please connect to the server using your encs credentials
- the app for this assignment runs on port 5257
- you, then, will find yourself in a "home page" with multiple links each of which 
refer to a question from the assignment, please click on the link of the question you
want to access, and there you are :)
